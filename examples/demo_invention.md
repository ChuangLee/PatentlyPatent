# 交底书：基于大语言模型的专利全流程辅助系统

## 技术领域
本发明涉及人工智能与知识产权的交叉领域，具体涉及一种基于大语言模型（LLM）对中国专利全流程
（检索、挖掘、撰写、OA 答辩）进行辅助的系统与方法。

## 要解决的技术问题
现有专利撰写工具：(1) 依赖人工，代理师效率低；(2) 商用 AI 系统将 LLM 当作模板填充器，缺少
法条嵌入式判断；(3) 缺乏 R20.2 必要技术特征最小集校验、A26.3/26.4 支持性校验、A33 修改超
范围锚定校验等中国实务核心规则。

## 技术方案
1. 通过 LLM 对交底文本做四要素拆解（领域 / 问题 / 手段 / 效果），每要素扩展同义词、上位、下位、英文。
2. 用关键字模式访问多个专利数据源（CNIPA、Google Patents、EPO OPS、USPTO ODP），并行召回、按 family 去重。
3. 基于检索结果与交底，生成 3 档不同概括度的独立权利要求（强 / 中 / 弱），每档附概括度风险标注。
4. 生成 5 档从属权利要求梯度（限定核心 / 可选特征 / 数值范围 / 优选实施例 / 解释性从权），网状引用。
5. 生成说明书五段骨架（技术领域 / 背景 / 发明内容 / 附图说明 / 具体实施方式）。
6. 在生成与改稿过程中嵌入法条 lint：R20.2、A26.3、A26.4、A33、R34，标注 high/mid/low 风险。

## 技术效果
- 资深代理师水平的初稿产出，端到端 ≤ 10 分钟；
- 法条 lint 命中率 ≥80%；
- 独权概括度等敏感决策提供候选 + 风险，由人审终判；
- 系统形态为 Python CLI + Claude Skill 薄包装，二者皆可独立使用。
