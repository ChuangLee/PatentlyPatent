# PatentlyPatent 使用手册

> 企业自助 AI 专利挖掘平台。访问 https://blind.pub/patent

## 5 步出交底书

| 步 | 操作 |
|---|---|
| 1 | 登录（账密或 CAS SSO） |
| 2 | 「新建报门」填发明描述，可拖入 PDF / PPT 作为附件 |
| 3 | 进项目工作台，AI 代理人**自动开工**调研（智慧芽 + Web + 知识库 + 你上传的资料） |
| 4 | AI 在聊天里问你 ≤3 个关键事实/数据，**你直接回复** |
| 5 | 信息足够后 AI 自动写 5 节交底书初稿 → 点右上「🎯 生成交底书 .docx」导出 |

## 工作台三栏

```
左：项目 + 文件树 | 中：聊天 | 右：文件预览（点文件树自动出，可全屏/钉住/关闭）
```

## 文件夹

| 文件夹 | 内容 | 谁能改 |
|---|---|---|
| 📁 **我的资料** | `0-报门.md`（系统自动落地）+ 你上传的文件 | 你 |
| 📁 **AI 输出** | AI 写的 5 节初稿 + `调研下载/` 关键文献摘要 + `专利交底书.docx` | AI（你也可编辑） |
| 📖 **本系统文档** | 使用手册 / PRD / HLD / 部署手册 | 只读 |
| 📚 **专利知识** | 419 篇 CN 实务知识库 | 只读 |

## 看 AI 在干啥

聊天里依次出现：
- **📋 当前计划**（顶部，6-8 步 TODO 实时勾选）
- **💭 思考卡片** —— "正在阅读资料 / 核查赛道 / 判断成熟度"
- **🔧 工具卡片** —— `search_patents` / `WebSearch` / `search_kb` / `read_user_file` / `save_research`，含入参 + 结果 + 耗时
- **AI 气泡** —— token 级流式输出 markdown（含 mermaid 图）

## 常见问题

| 问题 | 处理 |
|---|---|
| AI 没读我上传的文件 | 等附件全部上传完（顶部 thinking 提示完成）AI 才启动；如卡住强刷 |
| docx 按钮灰着 | AI 还没宣告"可以下笔了"；等收尾问题答完按钮会呼吸高亮 |
| AI 长时间没响应 | 60s 自动取消并提示重试；如重复发生联系运维查 CLI 凭证 |
| 想推翻重挖 | 在聊天里说"重新调研一下"即可 |

## 注意

- 不要上传敏感个人/客户数据
- AI 写的 5 节是**初稿**，必须代理人最终审
- 日预算 $10 上限，超了 503
