# 调研 3：中国资深专利代理师专家心智模型

> 资料：CNIPA《专利审查指南》2023 修订（2024.1.20 施行）、2025 代理师考试大纲、知产力/中伦/观韬/品源 2024-2025 实务文章

## 1. 检索阶段

**心法：检索不是"找到相同的"，而是"找到最能杀死本申请的"。**

- **四要素分析法**：技术领域/应用对象 + 要解决的技术问题 + 技术手段（核心+辅助特征）+ 技术效果。每要素出 3-5 个同义/上下位/中英日表达，做笛卡尔积。资深功夫在"上位词的胆量"。
- **漏斗式检索式**：
  - 第一层：核心关键词 AND IPC/CPC 主分类 → 200-500 命中
  - 第二层：F-term（机械/化学）+ DWPI Manual Code（化学/机械再分类）
  - 第三层：引文滚雪球（被引/施引/同族）+ forward/backward citation chasing
- **Markush（化学）**：识别通式权要覆盖边界，落到 substituent overlap
- **检索报告自检**：① X 类（破新颖性）有？ ② Y 类组合是否显而易见？ ③ 中/美/欧/日/韩+WO 五局覆盖 ④ 非专利文献（IEEE/arxiv/标准/手册）⑤ recall/precision 自评
- **LLM 自动化**：要素拆解、同义词扩展、IPC/CPC 推荐、检索式生成、命中摘要+对照表 ✅
- **人审保留**：上位激进度、Y 类组合合理性、Markush 边界

## 2. 技术挖掘 / 交底书

**心法：发明人讲"我做了什么"，代理师听出"为什么这样做、还能怎么做、不这样做会怎样"。**

- **问题树（5-Why + 1-What-if）**：技术问题 → 现有技术为何不行 → 核心机制 → 关键参数从哪来 → 替代手段 → 对手少做/多做/换做一步会怎样（→ 规避口）
- **上位/下位梯度**：核心写最上位（独权），次级写中位（从权一队），具体参数写最下位（末梯+实施例）。**独权能撑多大就撑多大，但不能撑到容易被无效**
- **效果挖掘**：一手段对应多效果（性能/成本/可靠性/能耗/可解释性…）→ OA 三步法第二步全靠它
- **规避反推**："省哪个特征还能 work？换什么等效？" → 等效项进从权或并列独权
- **Portfolio**：核心（产品独权）+ 方法独权 + 应用/系统独权 + 外围 + 改进型 + 防御性公开。**产品-方法-应用以产品为主**（CN 维权易）
- **LLM 自动化**：问题树展开、上位词候选、效果矩阵、规避点穷举 ✅
- **人审保留**：哪些是真发明点、上位是否撑得住 26.4、portfolio 战略对齐

## 3. 撰写——CN 实务"金标准"

### 权利要求布局
- **独权概括度**：再上一级被 X 类对比文件击破那一档；技术特征 7±2 是甜区
- **从权梯度**：① 进一步限定核心 ② 增可选特征 ③ 数值范围 ④ 优选实施例 ⑤ "解释性从权"（无效时锚回实施例）
- **多个独权（R34 单一性）**：产品+方法+用途+系统 经典四件套；并列独权之间须有"相同/相应特定技术特征"
- **引用关系**：网状而非线状

### 三段式 vs 非三段式
- 机械/结构 → 三段式（前序+特征部分）
- 软件/算法/化学 → 非三段式整体撰写
- **附图标号写说明书不写权要**（写进会被解释为限定）

### 说明书五段心法
- 背景技术：只讲现有技术做了什么+缺陷，不贬损过度
- 发明内容：与权要一一对应，每独权配技术效果段
- 具体实施方式：≥1 完整实施例 + 多变形例 + 参数端点和优选值，**A26.3 充分公开**靠这里
- **雷区**：① 功能性限定无实施例（A26.4 不支持）② 数值范围只给端点 ③ "等"字滥用 ④ 实施例与权要范围不匹配

### CN 特色
- **R20.2 必要技术特征**：独权须记载"解决问题所必需全部特征"——遗漏=驳回，多了=保护范围窄。心法：**最小可工作集**
- **发明构思**：贯穿全文的灵魂

### LLM 自动化
✅ 从权梯度、说明书五段骨架、实施例参数表、附图标号一致性
人审保留：独权概括度终判、必要特征认定、三段式区别特征划分、A26.3/26.4 风险

## 4. OA 答辩

### 创造性 A22.3 三步法（2023 指南强化）
1. **抢"最接近现有技术"定义权**：审查员选错 D1 是常见破绽——质疑领域/问题/效果是否真"最接近"
2. **重新认定实际解决的技术问题**：禁"事后诸葛亮"，基于"区别特征**在本申请中**所达到的效果"重新表述，"功能上相互支持的特征作为整体"
3. **打技术启示**：D1+D2 是否有 motivation？反向教导（teaching away）？公知常识须审查员举证（2023 新增）

### 其他答辩点
- **新颖性**：单篇对比、不能拼凑；上下位（下位破上位，上位不破下位）；惯用手段直接置换是隐蔽杀招
- **A26.3 公开充分**：补实验数据/参数；只能引申请日前公知文献
- **A26.4 支持**：权要"收"到实施例覆盖；功能性限定→结构+功能 或加从权兜底
- **A33 修改超范围（最大雷区）**：**只做删/缩，慎做加/改**；从实施例往独权"提特征"逐字检查；"中间概括"几乎必死
